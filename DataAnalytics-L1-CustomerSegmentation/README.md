# Customer Segmentation using RFM Analysis and K-Means Clustering

## Objective

Segment customers into meaningful groups using RFM analysis and K-Means clustering to help businesses improve marketing strategies.

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset

Online Retail Dataset

## Workflow

1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. RFM Analysis
5. Data Scaling
6. Elbow Method
7. K-Means Clustering
8. Customer Profiling
9. Business Insights

## Output

Customer clusters with business recommendations.
