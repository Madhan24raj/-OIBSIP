# Retail Sales Analysis Dashboard

## Objective
Perform Exploratory Data Analysis (EDA) on a retail sales dataset using Python and create an interactive Power BI dashboard.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Dashboard Features
- Total Sales KPI
- Total Orders KPI
- Average Sales KPI
- Total Quantity KPI
- Sales by Gender
- Sales by Product Category
- Quantity Sold by Category
- Sales Trend
- Age Distribution
- Interactive Slicers
